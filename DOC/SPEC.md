# 实习能量站（Intern Growth OS）


---

# 1. 产品概述

## 1.1 一句话定义

面向企业实习生培养场景的 **AI 成长导航与带教协同平台**。

## 1.2 核心价值

让实习生成长过程：**可视化 · 可预测 · 可干预**

## 1.3 目标用户

| 角色 | 核心目标 | 关注点 |
|------|----------|--------|
| 实习生 | 快速成长，成功转正 | 我该学什么？我成长到哪了？我离转正还有多远？ |
| 带教导师 | 高效带教，提升培养质量 | 怎么带？谁需要帮助？谁成长最快？ |
| 部门主管 | 识别高潜人才，追踪成长，分享知识 | 谁值得留下？谁需要关注？如何培养？ |
| 管理员 | 管理用户、分配导师、设置技能树 | 系统正常运行 |

## 1.4 角色配色方案

| 角色 | 主色 | 浅色背景 | 深色 |
|------|------|----------|------|
| 实习生 | #F5A623（橙色） | #FFF8E1 | #E09000 |
| 带教导师 | #4CAF50（绿色） | #E8F5E9 | #388E3C |
| 部门主管 | #9C27B0（紫色） | #F3E5F5 | #7B1FA2 |
| 管理员 | #607D8B（灰蓝） | #ECEFF1 | #455A64 |

# 2. 全局功能

## 2.0 系统主页（Landing Page）

**页面地址**：`/`

**功能**：
- 系统名称、标语和介绍
- 四大核心功能展示：游戏化成长、六维能力评估、技能树系统、AI成长助手
- 四种角色登录入口卡片：实习生、带教导师、部门主管、管理员
- 所有按钮在同一水平线上对齐

## 2.1 登录注册

### 登录页（`/login?role=xxx`）
- 邮箱 + 密码登录
- 角色身份验证（账号角色必须与登录入口匹配）
- 登录成功后跳转到对应角色的首页

### 注册页（`/register?role=xxx`）
- 填写邮箱、密码、姓名、部门、岗位
- 仅对实习生、导师、部门主管开放注册
- **管理员账号不开放注册**，由系统预置（admin@growth.com / admin123）

## 2.2 全局导航

侧边栏统一布局，包含：
- 系统品牌标识
- 个人头像（首字母）+ 姓名 + 角色标签（带角色配色）
- 根据角色动态显示导航菜单
- 退出登录按钮

# 3. 实习生功能

## 3.1 个人仪表盘（`/intern/dashboard`）

**核心模块：**

### A. Growth Twin 数字分身
- 当前等级（Lv.1 - Lv.10）和称号
- 经验值进度条（XP / 升级所需）
- 个人头像 + 基本信息

### B. 六维能力雷达图
六个评估维度：技术能力、沟通协作、业务理解、主动成长、问题解决、工程素养

### C. 技能树
- 按岗位匹配显示（精确 → 模糊 → 全部 三级匹配）
- 已解锁/未解锁状态
- 显示所需等级和XP奖励

### D. 成长时间轴
- 最近的成长事件（任务完成、升级、解锁等）

### E. 最近任务
- 显示最近5个分配的任务

## 3.2 成长分析（`/intern/growth-map`）

- 当前阶段概览（等级、XP）
- AI 转正预测
- 优势与待提升维度分析
- 完整六维雷达图
- 技能树详情
- AI 成长助手对话框（DeepSeek API）

## 3.3 成长任务（`/intern/tasks`）

- 任务列表（分状态筛选：全部/进行中/已完成/已逾期）
- 任务详情页：标题、描述、难度、XP、截止日期
- 提交工作内容
- 查看导师评价

## 3.4 成长周记（`/intern/weekly-report`）

**填写新周报（折叠面板）**：
- 默认折叠，点击展开
- 周数按入职时间自动计算（不可手动选择）
- 自评分（1-5星）+ 工作内容
- 提交后自动收起
- 当前周已提交时显示提示

**周报历史**：
- 每条独立可展开
- 展开后显示完整内容：
  - 📝 周报内容
  - 💬 导师评语
  - 💬 导师反馈（含⭐评分 + 六维评分详情）
  - 🎯 主管建议

## 3.5 评分历史（`/intern/score-history`）

- 当前六维评分概览
- 评分变更记录表
- 时间倒序排列

## 3.6 学习资料（`/intern/learning`）

- 部门主管发布的学习资料
- 按发布时间倒序
- 分类标签（技术/业务/沟通/成长/综合）

# 4. 带教导师功能

## 4.1 导师驾驶舱（`/mentor/dashboard`）

**模块：**
- 我的实习生卡片列表（显示等级和状态）
- 团队概览数据（总数、平均等级、完成率、关注数）
- 待办事项（待批阅周报、逾期任务）- 可滚动 + 点击确认完成
- 快速操作面板

## 4.2 任务管理（`/mentor/tasks`）⭐ 新增

**创建任务（折叠面板）**：
- 任务标题、描述、难度（1-5星）、XP奖励、截止日期
- 选择分配的实习生（仅自己带教的）
- 一键创建并自动分配

**任务列表**：
- 按状态筛选标签栏
- 不同状态颜色标识（待开始/进行中/已完成/已逾期）
- 显示完整任务信息和实习生
- 可删除已分配的任务

## 4.3 反馈管理（`/mentor/feedback`）

**写新反馈（折叠面板）**：
- 选择实习生、反馈类型（周度/里程碑/即时）
- 反馈内容
- 整体评分（1-5星）
- 六维能力评分（1-100分）

**反馈历史**：
- 列表展示所有已发反馈
- 点击展开查看完整内容 + 整体评分 + 六维评分
- 按反馈对象、时间筛选

## 4.4 培养计划（`/mentor/plan`）

**每位实习生的卡片：**
- 头像（48px圆形）+ 姓名 + 等级 + 岗位
- 三项统计：经验值、任务完成率、平均评分
- 六维能力雷达图 + 各维度分数卡片
- 「生成培养计划」按钮 → 调用 AI 生成针对性计划

**AI 生成的计划包含：**
- 当前能力分析
- 优势与不足
- 短期目标（1-2周）
- 中期目标（1个月）
- 长期目标（3个月）
- 具体行动建议

## 4.5 AI 带教助手（`/mentor/ai-copilot`）

- 实时对话框（与实习生 AI 助手样式一致）
- 调用 DeepSeek API
- 专门的导师角色系统提示词
- 用户消息（右侧绿色）+ AI 回复（左侧白色）
- 加载动画

## 4.6 实习生详情（`/mentor/intern/<id>`）

- 该实习生的完整成长档案
- 六维能力雷达图
- 任务记录、反馈记录、周报
- 评分历史

# 9. 核心业务流程

## 9.1 实习生完整成长流程

```
① 实习生注册 → 选择"实习生"角色 → 填写部门/岗位
   ↓
② 管理员分配导师
   ↓
③ 系统按岗位匹配技能树模板
   ↓
④ 进入仪表盘 → 看到 Growth Twin（Lv.1）+ 技能树
   ↓
⑤ 导师创建并分配成长任务 → 实习生执行
   ↓
⑥ 完成任务 → 获得 XP → 等级提升
   ↓
⑦ 达到技能解锁条件 → 技能树节点自动解锁
   ↓
⑧ 导师写反馈 → 六维能力评分更新 → 雷达图变化
   ↓
⑨ 实习生每周提交周报（按入职时间自动计算周数）
   ↓
⑩ 导师批阅 + 部门主管给出建议
   ↓
⑪ AI 生成成长分析 + 转正预测
   ↓
⑫ 持续循环 ⑤-⑪ → 等级提升 → 能力成长
   ↓
⑬ Lv.10 + 转正概率高 → 进入转正评审
```

## 9.2 导师带教流程

```
① 登录 → 驾驶舱 → 查看实习生列表
   ↓
② 创建任务 → 分配给特定实习生
   ↓
③ 周末批阅周报 → 写评语
   ↓
④ 写反馈 → 六维评分（影响实习生雷达图）
   ↓
⑤ 困惑时 → AI 带教助手获取建议
   ↓
⑥ 定期 → 「生成培养计划」→ AI 输出个性化方案
```

## 9.3 部门主管流程

```
① 登录 → 部门概况 → 查看部门数据
   ↓
② 周报审阅 → 看到部门内所有周报和导师评语
   ↓
③ 添加主管建议 → 实习生和导师都能看到
   ↓
④ 高潜人才识别 → 重点培养
   ↓
⑤ 发布学习资料 → 同步到实习生学习资料页
```

## 9.4 管理员流程

```
① 登录（admin@growth.com / admin123）
   ↓
② 用户管理 → 查看所有用户 → 给实习生分配导师
   ↓
③ 技能树管理 → 选择岗位 → 添加技能节点
   ↓
④ 实习生注册后自动按岗位匹配技能树
```

# 10. 数据库设计

## 10.1 核心表

| 表名 | 说明 |
|------|------|
| `users` | 用户基本信息（含角色） |
| `intern_profiles` | 实习生档案（关联导师、等级、XP） |
| `skill_dimensions` | 六个能力维度定义 |

| `skill_tree_templates` | 技能树模板（按岗位） |
| `skill_unlocks` | 实习生技能解锁记录 |
| `growth_tasks` | 任务定义 |
| `task_assignments` | 任务分配记录 |
| `weekly_reports` | 周报（含导师评语+主管建议） |
| `feedbacks` | 导师反馈（含六维评分） |
| `growth_logs` | 成长事件日志 |
| `learning_materials` | 学习资料 |
| `ai_analysis_cache` | AI 分析缓存 |

## 10.2 weekly_reports 表关键字段

```sql
mentor_comment       -- 导师评语
mentor_reviewed_at   -- 导师评语时间
leader_comment       -- 主管建议（V5新增）
leader_reviewed_at   -- 主管建议时间
```

# 13. 竞争优势

| 传统方案 | 我们的方案 |
|----------|------------|
| Excel 管理实习生成长 | 游戏化 Growth Twin 可视化 |
| 周会口头反馈 | 持续数据追踪 + AI 分析 |
| 转正靠主观判断 | AI 转正预测 + 量化指标 |
| 导师独自摸索 | AI Copilot 辅助带教 |
| 周报独立查看 | 周报+导师评语+主管建议三方协同 |
| 4 维度粗略评估 | 6 维度精细画像 |
| 无成长路径 | 岗位专属技能树（管理员可视化配置） |
| 任务靠口头分配 | 导师在线创建+分配+追踪 |
